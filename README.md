# Linux System Monitoring & Log Analysis Tool

This project is a Python-based Linux system monitoring tool that:
- Collects CPU and memory usage
- Parses system logs for ERROR and WARNING events
- Generates automated system health reports

## Technologies Used
- Python
- Linux (Ubuntu)
- System logs (/var/log/syslog)

## Use Case
Designed for system monitoring, research data collection, and academic projects.
